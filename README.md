# Ransomware Python Project

This project demonstrates how ransomware operates using Python, with a focus on educational and cybersecurity awareness purposes. The goal of this project is to simulate how malicious software can encrypt user data and restrict access to systems, helping learners understand the risks and mechanics behind ransomware attacks.

Ransomware is a type of malware that prevents users from accessing their files or operating systems until a ransom is paid. In this project, I implemented a simplified version that showcases key concepts such as encryption, decryption, password-based key generation, and file/folder protection.

The project walks through the process of:

- Generating encryption keys from user-defined passwords
- Encrypting and decrypting files and directories
- Handling errors and troubleshooting dependency issues
- Testing execution in different environments (Visual Studio Code vs. Kali Linux)

Through this hands-on implementation, this project highlights both the technical functionality of ransomware and the importance of ethical use and cybersecurity defense strategies.
